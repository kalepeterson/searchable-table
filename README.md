# Searchable Table - An Angular Component Library
Searchable Table is an Angular component designed to simplify displaying large amounts of data.

## User Guide
See the [package README file](projects/searchable-table/README.md) for guidance on using the library.

## Building

To build the project run:

```bash
ng build
```

This will compile your project and store the build artifacts in the `dist/` directory. By default, the production build optimizes your application for performance and speed.

## Running unit tests

To execute unit tests with the [Vitest](https://vitest.dev/) test runner, use the following command:

```bash
ng test
```
